== Description ==

### Cache:

This plugin caches all pages of the site, after caching, all the processing of the site will be reduced, leading to a considerable increase of performance.

### SEO:

This plugin also makes some optimizations to improve the way search engines and speed test tools fill the site as an example of Google Page Speed.

#### Some features

* Page Cache
* Minimizing for Javascripts (JS) files
* Minification for style files (CSS)
* Minimizing for HTML (HTML) files
* Browser Cache Enablement
* cache size limitation
* Gzip Compression
* Blocking / Releasing User Agent
* Control on Mobile and Desktop
* Block caching on certain pages
* Individual user cache
* Predefined Filters for PageSpeed

== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Cache SEO Speed'
3. Activate the 'Cache SEO Speed' from your Plugins page.
4. Go to Settings of 'Cache SEO Speed' on side menu.
