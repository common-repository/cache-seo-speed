foco em analisadores de desempenho

== Description ==

### Cache:

Este plugin armazena em cache todas as páginas do site, após o armazenamento em cache, todo o processamento do site será reduzido, levando a um aumento considerável de desempenho.

### SEO:

Esse plug-in também faz algumas otimizações para melhorar a maneira como os mecanismos de pesquisa e as ferramentas de teste de velocidade acessam o site, como exemplo o Google Page Speed.

#### Algumas funcionalidades

* Cache de páginas
* Minificação para arquivos Javascripts (JS)
* Minificação para arquivos de estilo (CSS)
* Minificação para arquivos HTML (HTML)
* Ativação de cache de navegador
* limitação de tamanho de cache
* Compressão Gzip
* Bloqueio/Liberação de User Agent
* Controle em Mobile e Desktop
* Bloquear cache em determinadas páginas
* Cache individual por usuários
* Filtros prédefinidos para PageSpeed

#### Licença

Cache SEO Speed é gratuito.

== Installation ==

=== From within WordPress ===

1. Acesse 'Plugins > Adicionar novo'
2. Procure por 'Cache SEO Speed'
3. Ative o 'Cache SEO Speed' na sua página de Plugins.
4. Vá para Configurações de 'Cache SEO Speed' no menu lateral.
